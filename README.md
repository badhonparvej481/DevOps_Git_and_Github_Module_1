## List of Useful Git Commands

## 1. Configuration

- Check if Git is installed
```shell
git --version
```

- Set user name
```shell
git config --global user.name "Your Name"
```

- Set email
```shell
git config --global user.email "your-email@example.com"
```

- Check configuration
```shell
git config --list
```
